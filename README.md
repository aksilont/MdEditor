## Установка
Проект генерируется с помощью [Tuist](https://tuist.io/).
Используемая версия - 3.40.0

**Шаги действий:**
```sh
git clone git@github.com:EncodedTeam/MdEditor.git
cd MdEditor/App
```
Редактирование манифеста:
```sh
tuist edit
```
Генерация проекта:
```sh
tuist generate
```
